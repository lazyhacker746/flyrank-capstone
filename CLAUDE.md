# AI Development Rules

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Python (Flask/FastAPI)
- Database: SQL

## Coding Conventions
- Prefer functional React components and React Hooks.
- Use Tailwind utility classes strictly; avoid writing custom CSS unless absolutely necessary.
- Write modular, clean Python code with type hints.
- Keep database queries optimized and use parameterized SQL queries to prevent injection.
- Before suggesting a large refactor, ask for confirmation and explain the architectural reasoning.
